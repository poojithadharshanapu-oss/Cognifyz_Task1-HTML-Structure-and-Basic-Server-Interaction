# HTML Node.js Express Server

## Project Overview

This project is a simple full-stack web application developed using Node.js, Express.js, and EJS template engine.

The application demonstrates how to create an Express server, render dynamic web pages using EJS, handle user input through forms, and apply CSS styling using static files.

## Features

- Created an Express.js server
- Used EJS as a template engine for dynamic pages
- Implemented GET and POST routes
- Handled form data submission
- Connected frontend pages with backend logic
- Added custom CSS styling

## Technologies Used

- HTML5
- CSS3
- Node.js
- Express.js
- EJS (Embedded JavaScript)

## Installation and Setup

1. Clone the repository:
git clone <repository-url>


2. Navigate to the project folder:


cd Task1-HTML-Node-Express-Server


3. Install dependencies:


npm install


4. Start the server:


node server.js


5. Open your browser and visit:


http://localhost:3000


## How It Works

- The Express server handles incoming requests.
- The EJS templates generate dynamic HTML pages.
- User input from the form is processed using POST requests.
- CSS files are served from the public folder for styling.
